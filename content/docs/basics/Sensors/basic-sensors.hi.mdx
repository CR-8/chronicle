# बेसिक सेंसर और पहचान

रोबोट को दुनिया को समझने के लिए।

## सेंसर प्रकार

### अवरक्त (IR) सेंसर

```
प्रतिबिंब विधि:
├─ IR LED निकाल देता है
├─ फोटोडायोड प्रतिबिंब पकड़ता है
├─ करीब = मजबूत सिग्नल
└─ दूर = कमजोर सिग्नल

आवेदन:
✓ लाइन अनुसरण
✓ वस्तु पहचान
```

### अल्ट्रासोनिक (HC-SR04)

```
तरंग गति विधि:
├─ ध्वनि तरंग भेजें
├─ प्रतिबिंब का इंतजार करें
├─ समय = दूरी
└─ 2-400 cm तक

फायदे:
✓ बहुत सटीक
✓ अंधेरे में काम करता है
```

### प्रकाश सेंसर

```
LDR (Light Dependent Resistor):
├─ प्रतिरोध = प्रकाश में परिवर्तन
├─ सस्ता ($0.50)
├─ एनालॉग आउटपुट
└─ धीमा (100ms)
```
