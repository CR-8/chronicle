# एनकोडर और प्रतिक्रिया

रोबोट को सटीक गति के लिए।

## रोटरी एनकोडर

```
चतुष्कोण डिकोडिंग:

A: ────┐   ┌──────
       └───┘
       
B: ──┐   ┌───────
     └───┘

अनुक्रम:
├─ A+ B+ = आगे
├─ A+ B- = पीछे
├─ गिनती = पूर्ण टर्न ट्रैक करता है
└─ ~1000 अलग-अलग प्रति टर्न
```

## ओडोमेट्री

```
दूरी = एनकोडर_गिनती × पहिया_परिधि

उदाहरण:
├─ 100 काउंट्स
├─ पहिया व्यास: 0.1 m
├─ परिधि: 0.314 m
├─ दूरी: 100 × 0.314 = 31.4 m यात्रा!
```

## प्रतिक्रिया नियंत्रण

```
PID लूप:
Error = Target_speed - Actual_speed
Motor_power = P×Error + I×(sum Error) + D×(change Error)

लाभ:
✓ सटीक गति
✓ ढलान पर स्तर गति
✓ विकृतियों को संभालता है
```
