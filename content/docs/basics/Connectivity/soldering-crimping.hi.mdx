# सोल्डरिंग और क्रिम्पिंग

विश्वसनीय विद्युत कनेक्शन।

## सोल्डरिंग

```
प्रक्रिया:
1. लौह को टिन करें
2. संयुक्त को गर्म करें
3. सोल्डर लागू करें
4. ठंडा करने दें (हिलाएं नहीं!)

अच्छा संयुक्त: चमकदार, चिकना
बुरा संयुक्त: मंद, दानेदार
```

## क्रिम्पिंग

```
सेटअप:
├─ तार छीलें
├─ क्रिम्प में डालें
├─ क्रिम्पर दबाएं
└─ पूर्ण

लाभ:
✓ तेज़ (सोल्डरिंग से)
✓ पुनरावृत्ति योग्य
✗ उपकरण की आवश्यकता
```

- कौशल की आवश्यकता है

**क्रिम्पिंग:**

- उपकरण निवेश
