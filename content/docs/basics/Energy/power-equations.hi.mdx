# शक्ति समीकरण और दक्षता

रोबोट की शक्ति प्रबंधन उसके ऑपरेटिंग समय को निर्धारित करती है।

## शक्ति समीकरण

### डीसी शक्ति

$$P = V \times I$$

जहाँ:
- P = शक्ति (वाट, W)
- V = वोल्टेज (वोल्ट)
- I = करंट (एम्पियर)

### प्रतिरोध में शक्ति

$$P = I^2 \times R = \frac{V^2}{R}$$

```
उदाहरण:
├─ 5A 10Ω प्रतिरोध में:
├─ P = 5² × 10 = 250 W
└─ बहुत गर्म हो जाएगा!
```

## ऊर्जा गणना

ऊर्जा = शक्ति × समय

$$E = P \times t$$

### बैटरी संधारण क्षमता

```
वाट-घंटे (Wh):
├─ 12V बैटरी 5 Ah
├─ ऊर्जा = 12V × 5 Ah = 60 Wh
└─ 20W उपकरण 3 घंटे तक चल सकता है
```

## दक्षता (Efficiency)

वास्तविक उपयोगी शक्ति / इनपुट शक्ति अनुपात:

दक्षता = (आउटपुट शक्ति / इनपुट शक्ति) × 100%

### मोटर दक्षता

```
छोटी DC मोटर: 70-80%
बड़ी DC मोटर: 80-90%
ब्रशलेस मोटर: 85-95%
गियरबॉक्स: 90-95% (प्रति चरण)
चेन ड्राइव: 95%+
बेल्ट ड्राइव: 90-95%
```
