# करंट, वोल्टेज और प्रतिरोध

विद्युत चार मूल अवधारणाओं पर आधारित है जो सभी रोबोट सिस्टम को शक्ति देती हैं।

## ओम का नियम

सबसे महत्वपूर्ण विद्युत समीकरण:

$$V = I \times R$$

जहाँ:
- V = वोल्टेज (वोल्ट, V)
- I = करंट (एम्पियर, A)
- R = प्रतिरोध (ओम, Ω)

## करंट (Current)

विद्युत आवेश का प्रवाह:

```
जलप्रवाह सादृश्य:
├─ पानी बहता है = करंट बहता है
├─ अधिक दबाव = अधिक प्रवाह
└─ अधिक वोल्टेज = अधिक करंट

सूत्र: I = V / R
```

### करंट यूनिट

```
एम्पियर (A):
├─ 1 A = 1 कूलम्ब / सेकंड
├─ 1000 mA = 1 A
├─ 1000 µA = 1 mA
└─ रोबोटिक्स: आमतौर पर 0.5-5 A
```

### करंट उदाहरण

```
USB: 0.5 A
रोबोट मोटर: 2 A
12V 50W पावर सप्लाई: 4.2 A
विशाल रोबोट आर्म: 10-20 A
```

## वोल्टेज (Voltage)

विद्युत दबाव जो करंट को धकेलता है:

```
जल सादृश्य:
├─ ऊंचाई अंतर = वोल्टेज अंतर
├─ अधिक अंतर = अधिक बल
└─ पानी बहता है = करंट बहता है

सूत्र: V = I × R
```

### वोल्टेज स्रोत

```
AA बैटरी: 1.5 V
9V बैटरी: 9 V
USB पावर: 5 V
रोबोटिक्स शक्ति: 12 V (सामान्य)
उद्योग: 24 V या 48 V
```

### वोल्टेज ड्रॉप

तार में वोल्टेज खो जाता है:

```
सूत्र: V_drop = I × R_wire

उदाहरण:
├─ 5 A करंट, तार R = 0.1 Ω
├─ V_drop = 5 × 0.1 = 0.5 V
└─ 12V बैटरी → 11.5V मोटर पर (0.5V खो)

लंबा तार = अधिक प्रतिरोध = अधिक ड्रॉप
```

## प्रतिरोध (Resistance)

करंट को रोकता है:

```
जल सादृश्य:
├─ संकीर्ण पाइप = प्रतिरोध
├─ अधिक संकीर्ण = अधिक बाधा
└─ पानी कम बहता है
```

### सामान्य प्रतिरोध

```
तांबे का तार (अच्छा): 0.00001 Ω प्रति मीटर
एल्यूमीनियम (अच्छा): 0.00003 Ω प्रति मीटर
निक्रोम तार (गरम करने के लिए): 1 Ω प्रति मीटर
LED (अवरोधक): 10-1000 Ω
मोटर (ठंडा): 1-10 Ω
```

---

## विद्युत शक्ति

शक्ति = वोल्टेज × करंट

$$P = V \times I$$

### शक्ति गणना

```
12V 5A लोड:
├─ P = 12 V × 5 A = 60 W
└─ यह 60 वाट बिजली खींचता है

5V USB उपकरण:
├─ P = 5 V × 1 A = 5 W
└─ कम शक्ति

बैटरी क्षमता:
├─ 1000 mAh @ 12V = 12 Wh
├─ यह 60W उपकरण को 12 मिनट तक चला सकता है
└─ (12 Wh / 60 W = 0.2 घंटे)
```

---

## ऊर्जा खपत

विद्युत ऊर्जा = शक्ति × समय

$$E = P \times t$$

### ऊर्जा यूनिट

```
वाट-घंटे (Wh):
├─ 60W उपकरण 1 घंटे के लिए = 60 Wh
├─ 60W उपकरण 2 घंटे के लिए = 120 Wh
└─ रोबोटिक्स में सामान्य

किलोवाट-घंटे (kWh):
├─ 1 kWh = 1000 Wh
└─ विद्युत बिल गणना के लिए

जूल (J):
├─ 1 Wh = 3600 J
└─ सैद्धांतिक गणना के लिए
```

## श्रृंखला बनाम समानांतर

### श्रृंखला (Series)

```
एक के बाद एक:

─ + ─────────(─R1─)────────(─R2─)───────── −

वोल्टेज जोड़ें:
├─ कुल V = V1 + V2

करंट समान:
├─ I1 = I2 = I कुल

प्रतिरोध जोड़ें:
├─ R_total = R1 + R2

उदाहरण (2 × 12V बैटरी):
├─ कुल: 24V
└─ यदि एक बैटरी 5A देती है: अभी भी 5A
```

### समानांतर (Parallel)

```
सभी एक साथ जुड़े:

            (─R1─)
            │   │
─ + ────┬───┤   ├───┬────── −
        │  (─R2─)   │
        └─────┬─────┘

वोल्टेज समान:
├─ V1 = V2 = V कुल

करंट जोड़ें:
├─ I_total = I1 + I2

प्रतिरोध घटाएं:
├─ 1/R_total = 1/R1 + 1/R2

उदाहरण (2 × 5A पावर सप्लाई):
├─ अभी भी 12V
└─ लेकिन 10A कुल (दोगुना करंट!)
```

## शार्ट सर्किट खतरा

<Callout type="warn" title="शार्ट सर्किट सुरक्षा">

⚠️ यदि प्लस सीधे माइनस से जुड़ा हो (R = 0):

सूत्र: I = V / R = V / 0 = ∞ (अनंत!)

वास्तविकता:
├─ कुछ सेकंड में तार जल जाता है
├─ बैटरी विस्फोट हो सकती है
├─ घर की आग लग सकती है
└─ हमेशा फ्यूज/सर्किट ब्रेकर स्थापित करें

सुरक्षा:
✓ सभी शक्ति लाइनों पर फ्यूज
✓ तार को डबल-चेक करें
✓ शार्ट सर्किट से पहले अलग करें
✓ पहले प्रोटोटाइप करें

</Callout>

## रोबोटिक्स उदाहरण

### 2-मोटर रोबोट

```
सेटअप:
├─ 12V बैटरी
├─ 2 × DC मोटर @ 2A प्रत्येक
├─ माइक्रोकंट्रोलर @ 0.5A
├─ सेंसर @ 0.3A
├─ कुल: 5.1 A

बैटरी क्षमता:
├─ 5000 mAh = 5 Ah
├─ ऊर्जा = 12V × 5 Ah = 60 Wh
├─ समय = 5 A कुल / 5 Ah = 1 घंटा
└─ 60 मिनट चलेगा
```

### शक्ति वितरण

```
12V 60W बैटरी:

          12V बैटरी
            │
       ┌────┴────┐
       │          │
    फ्यूज     फ्यूज
     20A       10A
       │          │
     मोटर    कंट्रोलर
     40W        10W
       │          │
     2.5A       0.8A
```

## सारांश

**ओम का नियम:**
- V = I × R
- करंट, वोल्टेज, प्रतिरोध को जोड़ता है

**करंट:**
- विद्युत प्रवाह
- एम्पियर (A) में मापा जाता है
- रोबोटिक्स: 0.5-10 A

**वोल्टेज:**
- विद्युत दबाव
- वोल्ट (V) में मापा जाता है
- रोबोटिक्स: 5V, 12V, या 24V

**शक्ति:**
- P = V × I
- वाट (W) में मापा जाता है
- बैटरी जीवन निर्धारित करता है
