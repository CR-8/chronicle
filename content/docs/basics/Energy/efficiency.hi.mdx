# ऊर्जा दक्षता

कुशल रोबोट लंबे समय तक चलता है।

## ऊर्जा हानि स्थान

```
10 W इनपुट शक्ति:

मोटर नुकसान: 2 W (20%) → घर्षण, गर्मी
गियरबॉक्स नुकसान: 0.5 W (5%)
ड्राइवट्रेन नुकसान: 0.3 W (3%)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
उपयोगी: 6.7 W (67%)

कुल दक्षता: 67%
```

## नुकसान कम करना

```
✓ ब्रशलेस मोटर (90%+)
✓ चिकनाई लगाएं (घर्षण कम)
✓ अल्टीमेट गियर (कम चरण)
✓ हल्का निर्माण
✓ कम रोलिंग प्रतिरोध टायर
```
