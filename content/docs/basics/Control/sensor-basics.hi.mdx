## ADC (Analog-to-Digital Converter)

```
एनालॉग सिग्नल 0-5V:
└─ 0V → 0
└─ 2.5V → 512
└─ 5V → 1023

Arduino analogRead():
int value = analogRead(A0);  // 0-1023
```
