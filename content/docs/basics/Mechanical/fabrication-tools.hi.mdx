# निर्माण और उपकरण

रोबोट भाग बनाना।

<Callout type="warn" title="कार्यशाला सुरक्षा">

⚠️ सुरक्षा नियम:
- हमेशा आंखें बंद करें
- गतिशील भागों से दूर हाथ रखें
- काम को क्लैम्प करें (हाथ नहीं)
- तीव्र गति वाले उपकरणों को कभी अकेला न छोड़ें

</Callout>

## उपकरण

```
न्यूनतम सेटअप:
├─ हाथ ड्रिल
├─ हैकसॉ
├─ फाइलें
└─ सुरक्षा चश्मा

पेशेवर:
├─ ड्रिल प्रेस
├─ कोण पीसने वाला
├─ वेल्डर
└─ धातु कूचनी
```

## सारांश

**ड्रिलिंग:**
- तीव्र गति, हल्का दबाव
- पायलट छेद पहले

**काटना:**
- तेज़ ब्लेड महत्वपूर्ण
- सामग्री के लिए सही गति

**सोल्डरिंग:**
- संयुक्त को गर्म करें, सोल्डर नहीं
- शांत होने तक हिलाएं नहीं
