# संचरण और गति

शक्ति को गति में परिवर्तित करना।

## गियर प्रकार

```
स्पर गियर: सरल, कुशल
├─ अनुपात: 2:1 से 50:1
├─ दक्षता: 95%+
└─ शोर: अधिक

ग्रहीय: कॉम्पैक्ट
├─ अनुपात: 3:1 से 100:1
├─ दक्षता: 92%
└─ शोर: कम

कृमि: उच्च अनुपात
├─ अनुपात: 5:1 से 100:1
├─ दक्षता: 60-70%
└─ स्व-लॉकिंग
```

## चेन बनाम बेल्ट

```
चेन:
✓ कोई पर्ची नहीं (धनात्मक)
✓ दूरी तक फैल सकता है
✗ शोर, रखरखाव

बेल्ट:
✓ चुप ऑपरेशन
✓ कोई रखरखाव नहीं
✗ फिसल सकता है
```
