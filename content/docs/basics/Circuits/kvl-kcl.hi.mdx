# किरचॉफ का नियम

सभी सर्किट विश्लेषण के लिए आवश्यक।

## किरचॉफ वोल्टेज नियम

एक लूप में वोल्टेज जोड़ 0 होता है।

सर्किट उदाहरण:

```
12V बैटरी - R1 - R2 - ग्राउंड

वोल्टेज समीकरण:
12V = VR1 + VR2
```

## किरचॉफ करंट नियम

एक नोड पर करंट इन = करंट आउट।

नियम: एक बिंदु में प्रवेश करने वाला कुल करंट उस बिंदु से निकलने वाले कुल करंट के बराबर होता है।

---

## सारांश

KVL: लूप का कुल वोल्टेज शून्य होता है
KCL: नोड पर करंट संरक्षित होता है
उपयोग: सर्किट समीकरण हल करना
