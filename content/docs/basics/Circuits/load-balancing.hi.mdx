# लोड संतुलन

सर्किट में सभी घटक संतुलित शक्ति चाहते हैं।

## वोल्टेज ड्रॉप

```
लंबे तार से नुकसान:

12V ──────[लंबा तार]──── मोटर

V_drop = I × R
यदि I = 5A, R = 0.1Ω
V_drop = 0.5V

मोटर को: 12V - 0.5V = 11.5V (कम!)
```

## समाधान

```
✓ बड़े तार (कम प्रतिरोध)
✓ अलग बैटरी (विभिन्न डिवाइस के लिए)
✓ क्षेत्र वितरण (स्थानीय विद्युत)
```

## सारांश

**लोड संतुलन:**
- सभी को पर्याप्त वोल्टेज दें
- तार आकार महत्वपूर्ण है
- अलग शक्ति आवश्यक हो सकती है
