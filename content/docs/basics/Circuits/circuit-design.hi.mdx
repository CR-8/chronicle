# सर्किट डिजाइन

व्यावहारिक सर्किट डिजाइन सिद्धांत।

## योजनाबद्ध संगठन

```
अच्छा डिजाइन:
├─ शक्ति बाईं ओर
├─ संकेत बीच में
├─ ग्राउंड दाईं ओर
└─ समूहीकृत कार्यक्षमता
```

## पीसीबी लेआउट

```
नियम:
✓ शक्ति/ग्राउंड व्यापक (मोटी रेखाएं)
✓ संकेत छोटे (पतली रेखाएं)
✓ क्रॉसिंग कम करें
✓ घटक को बारीकी से स्थान दें
```

---

## सारांश

**डिजाइन सिद्धांत:**
- सरल और मॉड्यूलर बनाएं
- शक्ति वितरण महत्वपूर्ण
- परीक्षण से पहले तीन बार जांच करें
