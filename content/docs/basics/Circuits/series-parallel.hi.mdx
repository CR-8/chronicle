# श्रृंखला और समानांतर सर्किट

## श्रृंखला (Series)

```
एक के बाद एक जुड़े हुए:

+ ─────[R1]─────[R2]─────[R3]───── −

वोल्टेज: V = V1 + V2 + V3
करंट: I = समान (सभी में)
कुल R: R_total = R1 + R2 + R3
```

## समानांतर (Parallel)

```
सभी एक ही बिंदु से जुड़े:

    [R1]
    [R2]
    [R3]

वोल्टेज: V = समान (सभी में)
करंट: I = I1 + I2 + I3
कुल R: 1/R_total = 1/R1 + 1/R2 + 1/R3
```

---

## रोबोटिक्स अनुप्रयोग

```
बैटरी:
└─ श्रृंखला: 2×6V = 12V (अधिक वोल्टेज)
└─ समानांतर: 2×5Ah = 10Ah (अधिक समय)

मोटर:
└─ श्रृंखला: सीमित (केवल प्रोटोटाइप)
└─ समानांतर: सामान्य (अधिक शक्ति)
```
