# मोटर नियंत्रण

डीसी मोटर को नियंत्रित करना पूरी रोबोटिक्स है।

## गति नियंत्रण

### वोल्टेज नियंत्रण

```
अधिक वोल्टेज = तेज़ गति

12V → 100% गति
6V → ~50% गति (रैखिक नहीं)
0V → बंद
```

### PWM (Pulse Width Modulation)

```
तेज़ चालू/बंद स्विचिंग:

100% PWM:  ─────┐  ─────┐  ─────┐
              └──┘  └──┘  └──┘

50% PWM:   ──┐  ──┐  ──┐  ──┐
            └──┘└──┘└──┘└──┘

25% PWM:   ─┐  ─┐  ─┐  ─┐
           └──┘└──┘└──┘└──┘

लाभ:
✓ ऊर्जा बर्बादी नहीं
✓ सुचारू नियंत्रण
✓ रोबोटिक्स मानक
```

## दिशा नियंत्रण

### H-ब्रिज

```
चार MOSFET स्विच:

      + 12V
        │
    ┌───┴───┐
    ├───┬───┤
    │ M │ N │
 ┌──┤   ├──┐
 │  │ O │ P│  
 │  │ T │ │─ मोटर
 │  │ O │ │
 │  │ O │ │
 │  │ R │ │
 │  │ R │ │
 │  │ A │ │
 │  │ Y │ │
 └──┤   ├──┘
    ├───┴───┤
    │  │  │
    ├───┼───┤
    └───┴───┘
      GND

नियंत्रण:
├─ M+, P- ON: आगे
├─ M-, P+ ON: पीछे
├─ सभी OFF: तटस्थ
└─ M+, P+ ON: ब्रेक (शॉर्ट)
```
