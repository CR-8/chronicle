# मोटर और ड्राइव गियर

रोबोट गति नियंत्रण के लिए।

## मोटर चयन

```
छोटा रोबोट: 6V DC, 0.5A, 100-300 RPM
मध्यम रोबोट: 12V DC, 2-5A, 100-500 RPM
बड़ा रोबोट: 24V DC, 10A+, 50-200 RPM

सामान्य नियम:
- RPM × टोक़ = शक्ति (मोटा)
- उच्च गति = कम टोक़
- कम गति = उच्च टोक़
```

## गियर अनुपात

```
10:1 गियर:
├─ 300 RPM मोटर → 30 RPM आउटपुट
├─ 0.5 N⋅m → 5 N⋅m (10× अधिक!)
└─ गति कम, शक्ति अधिक

1:1 (प्रत्यक्ष):
├─ 300 RPM मोटर → 300 RPM
├─ कम टोक़
└─ तेज़ गति
```
