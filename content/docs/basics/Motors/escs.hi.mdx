# इलेक्ट्रॉनिक गति नियंत्रक (ESC)

ड्रोन और ब्रशलेस मोटर के लिए आवश्यक।

## ESC कार्य

```
PWM इनपुट → ESC → तीन-चरण आउटपुट → ब्रशलेस मोटर

1000µs PWM: 0% (बंद)
1500µs PWM: 50% (मध्य)
2000µs PWM: 100% (पूर्ण)
```

## तीन-चरण आउटपुट

```
A, B, C तारों को चक्रीय क्रम में स्विच करें:

समय 0: A+, B-, C-
समय 1: A+, B+, C-
समय 2: A-, B+, C-
समय 3: A-, B+, C+
समय 4: A-, B-, C+
समय 5: A+, B-, C+

फ़्रीक्वेंसी: आमतौर पर 8-32 kHz
```

**ESC:**

- DShot बेहतर नियंत्रण के लिए
