# बैटरी रेटिंग

बैटरी लेबल समझें।

## मुख्य पैरामीटर

```
उदाहरण: 3S 5000mAh 50C 11.1V

3S = 3 कोशिकाएं श्रृंखला में = 11.1V
5000mAh = क्षमता = 5 Ah
50C = C-रेटिंग = 50 × 5000mA = 250A तक!
11.1V = नाममात्र वोल्टेज
```

## C-रेटिंग

```
अधिकतम निर्वहन = क्षमता × C-रेटिंग

उदाहरण:
├─ 5000mAh @ 50C
├─ अधिकतम = 5 Ah × 50 = 250A
└─ यह 5 सेकंड में सभी को खाली कर सकता है!
```

**बैटरी चयन:**

- C-रेटिंग अनुप्रयोग के लिए पर्याप्त
