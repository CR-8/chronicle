## Hello World

Welcome to the Chronicle blog! This is our first example post to demonstrate the blogging capabilities of our documentation site.

## What is Chronicle?

Chronicle is a comprehensive documentation platform built with Fumadocs and Next.js. It provides documentation for various robotics challenges including:

- **Autonomous** - Autonomous robot navigation and control
- **Combat** - Robot combat strategies and techniques
- **Drone** - Drone programming and flight control
- **Race** - Racing robot optimization
- **Soccer** - Robot soccer gameplay

## Why a Blog?

We're adding a blog to share:

- **Updates** - Latest features and improvements
- **Tutorials** - In-depth guides and walkthroughs
- **Case Studies** - Real-world implementation examples
- **Community Stories** - Success stories from our users
